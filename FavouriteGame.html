Counter-Strike: Global Offensive

This is a first person shooting game. I have dedicated more time to this game than i would like to admit. The game has recently gone free to play which has brought a lot more players to the game. The game is centered around two teams fighting over bomb sites, the two teams are terrorists and counter-terrorists. The objective for the terrorists is to try plant the bomb on the bombsite while the counter-terrorists must defend the bombsite and stop them.

Reasons I like the game

  • The game is very competitive which is very appealing to me.
  • The game also has a steep learning curve that rewards players that practice and have good muscle memory and reflex's.

Rocket League

Rocket League is a game where the player controls cars inside a stadium playing soccer with those cars. The object of the game is to score more goals than your opponent.

Reasons I like the game

  • The game is very competitive which is very appealing to me. The game is also very fun to play with friends.
  • The game also has a steep learning curve that allows the user to sink multiple hours into to try learn different mechanics.

Among us

Among us is a multiplayer social deduction game where the lobby can consist of anywhere from 4-10 players. The game has 3 different maps which consist of a a launch site, on a spaceship and finally on an alien planet. In all the maps the goal is the same, you are either a crew mate; this means you want to complete all your tasks or throw out the imposter, or you are the imposter who is trying to sabotage the game by killing all other players.

Reasons I like the game

  • The game can be a lot of fun playing with friends.
  • The game is all about either killing others or trying to find the killer before getting killed.
