Rocket League

Rocket League is a game where the player controls cars inside a stadium playing soccer with those cars. The object of the game is to score more goals than your opponent.

Reasons I like the game

  • The game is very competitive which is very appealing to me. The game is also very fun to play with friends.
  • The game also has a steep learning curve that allows the user to sink multiple hours into to try learn different mechanics.
